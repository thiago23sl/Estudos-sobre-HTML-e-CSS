Listas

Listas Ordenadas

para abrir uma liista voce usa a tag <ol> e para colocar um item na lista usa a tag <li>

  1. acordar
  2. tomar café
  3. escovar os dentes
  4. tomar banho
  5. se arrumar
  6. passar perfume
  7. sair para trabalhar

ul ou lista não ordenada

para fazer uma lista não ordenada se usa a tag <ul>

  • comer batata
  • lavar a batata
  • cair no soco com alguem

<ul type="square">

  • a

<ul type="circle">

  • a

<ul type="disc">
